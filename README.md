# 说明
从其他B站视频导入弹幕到当前视频，支持设置和弹幕列表管理

本项目使用dpskv4pro+cc生成 纯AI 除了构思方案外无任何人工成分

# 更新
1.1.0 ← 增加了对番剧链接的支持。

# 使用方法
1、浏览器安装油猴/暴力猴之类的脚本插件
2、直接把本项目的js文件丢进他们里面安装即可

# 功能
允许重复输入视频链接，脚本会直接叠加获取到的对应弹幕

# 应用场景
垃圾画质的动画很多弹幕

好心人传的4K分辨率动画 极少弹幕 此时他就有用了。
